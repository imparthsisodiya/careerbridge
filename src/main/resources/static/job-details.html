Job Description

We are looking for a Java Backend Developer who has strong knowledge of Java, Spring Boot, REST APIs, MySQL, and backend problem solving. The candidate should be able to build scalable APIs and work with databases efficiently.

  • Design clean and efficient REST APIs.

Java Spring Boot MySQL REST API Hibernate

  • Knowledge of Spring Boot and SQL databases.
  • Good communication and problem-solving skills.
